  • 8/2/2012

    La-kenak v2.0

    1.Προστέθηκε η δυνατότητα αποθήκευσης της τελευταίας μελέτης στη βάση δεδομένων.
    2.Προστέθηκε στον έλεγχο θερμομονωτικής επάρκειας ο πίνακας ανοιγμάτων στο τεύχος.
    3.Προστέθηκε η δυνατότητα προσθήκης x/y τοίχων/ανοιγμάτων χωρίς να εξαρτώνται από ένα συνολικό αριθμό (προηγούμενη έκδοση).
    4.Προστέθηκε η δυνατότητα επεξεργασίας του τεύχους (τμήματα αυτού που μεταβάλλονται).
    5.Τμήματα του τεύχους έχουν προστεθεί στη βάση δεδομένων για γρηγορότερη φόρτωση της σελίδας υπολογισμών.

  • 18/2/2012

    La-kenak v2.1

    Τροποποιήσεις απο tsak (kenakv2-mod-by-tsak.rar)

    1. Προσθήκη βιβλιοθήκης TCPDF
    2. Τροποποίηση υπολογισμού δομικών στοιχείων {προσθήκη javascript, stylesheets, φόρμας υπολογισμού, αποθήκευσης, ανάκτησης από βιβλιοθήκες, προσθήκη πινάκων} Προσθήκη πίνάκων: domika_floors, +2 columns στον πίνακα domika_toixoi
    3. Προσθήκη image_creation scripts {οριζόντιων, κατακόρυφων στοιχείων για την παραγωγή εικόνων}
    4. Προσθήκη pdf creation script για την παραγωγή εκτυπώσιμου φύλλου υπολογισμού δομικού στοιχείου
    5. Προσθήκη jquery για την επιλογή του τοίχου και την αποθήκευση.
    6. Προσθήκη functions (includes/functions.php)
    7. Προσθήκη php script για είσοδο των νέων πινάκων στη βάση
    Τροποποιήσεις απο sw0rdf1sh (kenakv2.1.rar)

    1. Προσθήκη υπολογισμού ισοδύναμου U για δάπεδο επί εδάφους
    2. Προσθήκη του πίνακα 9α της ΤΟΤΕΕ 20701-2 στη βάση δεδομένων για τον παραπάνω υπολογισμό
    3. Τροποποίηση του υπολογισμού ανοιγμάτων ώστε να βοηθήσει μετέπειτα για χρήση javascript και tcpdf.
    4. Χρήση του css από τον χρήστη tsak καθώς θεωρήθηκε λειτουργικότερο ως προς την προβολή των βιβλιοθηκών
    5. Προσθήκη γραφικών (Προέρχονται απο icon-archive.com)
    6. Διόρθωση πινάκων θερμογεφυρών ως προς τα αρνητικά πρόσημα συγκεκριμένων συντελεστών.
